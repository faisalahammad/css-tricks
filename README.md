#### Control Placeholder color
	class/id/null input::-webkit-input-placeholder, class/id/null textarea::-webkit-input-placeholder {
	  color: #fff;
	  opacity: 1;
	}
	
	class/id/null input:-moz-placeholder, class/id/null textarea:-moz-placeholder {
	  color: #fff;
	  opacity: 1;
	}
